<Intro>
  Destaca uma observação que o leitor não deveria passar batido, sem interromper a leitura.
</Intro>

```md
<Hint>
  Lembre de definir `MDX` antes de rodar o build.
</Hint>
```

resultado:

<Hint>
  Lembre de definir `MDX` antes de rodar o build.
</Hint>

Para avisos com peso semântico (nota, dica, alerta, perigo), prefira os
[alertas do GitHub](/authoring/gha), que já vêm com cor e ícone próprios.

<Intro>
  Gera um índice de todas as páginas da documentação, agrupadas pela seção a que pertencem. Útil
  como página inicial ou como mapa do site.
</Intro>

```md
<Entries />
```

resultado:

<Entries />

A lista é montada a partir dos arquivos, então ela nunca fica desatualizada: uma página nova aparece
aqui sozinha.

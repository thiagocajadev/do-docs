<Intro>
  O `Intro` é o parágrafo de abertura de uma página, renderizado maior que o corpo do texto. Serve
  para dizer, em duas ou três linhas, o que a página cobre.
</Intro>

```md
<Intro>
  Aponte o DoDocs para uma pasta de Markdown e ele devolve um site.
</Intro>
```

resultado:

<Intro>
  Aponte o DoDocs para uma pasta de Markdown e ele devolve um site.
</Intro>

Aceita Markdown dentro: **negrito**, `código` e [links](https://github.com/thiagocajadev/do-docs).

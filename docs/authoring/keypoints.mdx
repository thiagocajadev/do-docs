<Intro>
  Resume os pontos que a página aborda. Costuma vir logo depois do `Intro`, para o leitor decidir
  se é isso que ele procura.
</Intro>

```md
<Keypoints title="O que você vai aprender">
  <KeypointsItem>Primeiro item</KeypointsItem>
  <KeypointsItem>Segundo item, com **destaque**</KeypointsItem>
</Keypoints>
```

resultado:

<Keypoints title="O que você vai aprender">
  <KeypointsItem>Primeiro item</KeypointsItem>
  <KeypointsItem>Segundo item, com **destaque**</KeypointsItem>
</Keypoints>

O `title` é opcional. Cada `KeypointsItem` vira um marcador, e aceita Markdown.

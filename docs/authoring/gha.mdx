<Intro>
  Os [alertas do GitHub](https://github.com/orgs/community/discussions/16925) funcionam aqui, com a
  mesma sintaxe. Cada nível tem cor e ícone próprios, derivados do seu tema.
</Intro>

Escreva como uma citação, começando pelo nível:

```md
> [!NOTE]
> Informação que o leitor deve considerar, mesmo passando o olho.
```

resultado:

> [!NOTE]
> Informação que o leitor deve considerar, mesmo passando o olho.

Os cinco níveis:

> [!NOTE]
> Informação que o leitor deve considerar, mesmo passando o olho.

> [!TIP]
> Algo opcional, que ajuda o leitor a se sair melhor.

> [!IMPORTANT]
> Informação essencial para o leitor ter sucesso.

> [!WARNING]
> Conteúdo urgente, que pede atenção imediata.

> [!CAUTION]
> Risco real, com consequências negativas se ignorado.

As cores de cada nível são configuráveis: veja `theme_note`, `theme_tip`, `theme_important`,
`theme_warning` e `theme_caution` na [configuração](/getting-started/configuration).
